# Terraform

## Quick start

### Working cycle

```bash
mkdir <project-name>
cd <project-name>
touch main.tf
vim main.tf     # change main.tf file for your infrastructures
terraform init
terraform fmt
terraform validate
terraform apply
terraform show
terraform state list
terraform destroy
```
